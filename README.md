# Tuum
Estonian word for "core", it should be set of tools to build modern web apps. 
It's also an alternative build approach with esbuild for Vite.

## Installation
```shell
npm i
```

## Build (esbuild)
```shell
npm run build
```
or separately
```shell
npm run build:styles
npm run build:scripts
```

## Serve (vite)
```shell
npm run serve
```
